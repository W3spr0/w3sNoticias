GitHub tem receita de US$ 1 bilhão e 90 milhões de usuários

  • O código desta pagina foi feito do zero por W3spr0, se há algo identico com outro site é pura coincidencia

O popular serviço de repositório de código GitHub foi adquirido pela Microsoft há quatro anos, por US$ 7,5 bilhões. Agora, a empresa divulgou, na reunião de resultados, os novos dados de desempenho da plataforma, e os números são grandes. A receita anual do GitHub atingiu a marca de US$ 1 bilhão, bem acima dos US$ 200 a US$ 300 milhões quando foi comprado.


Essa é a primeira vez que a Microsoft divulga informações sobre o serviço, desde a aquisição. “O espírito de desenvolvedor do GitHub nunca foi tão forte. Mais de 90 milhões de pessoas agora usam o serviço para criar software para qualquer nuvem, em qualquer plataforma – até três vezes”, disse Satya Nadella, CEO da Microsoft, na teleconferência.


Os 90 milhões de usuários são aqueles ativos. Os números anteriores, do ano da compra, 2018, mostravam 28 milhões de pessoas no GitHub. Já em novembro de 2022, a quantidade saltou para 73 milhões.


A Microsoft evita interferir muito no GitHub. Inicialmente, houve um temor por parte dos desenvolvedores de que a big tech mudasse o modo de operação do serviço, reduzindo as ofertas gratuitas. Ate o momento, ocorreu o contrário, com a plataforma expandindo a parte sem custos e mantendo o código aberto.
